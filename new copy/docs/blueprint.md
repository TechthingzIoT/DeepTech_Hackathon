# **App Name**: AgriSense

## Core Features:

- User Authentication: Secure user sign-up, login, logout, and password reset functionality.
- IoT Node Management: Add, remove, and manage IoT nodes with name, location, sensor performance, and per-node secret keys.
- Real-Time Dashboard: Display sensor readings (temperature, humidity, soil moisture, water level, pump status) using interactive charts.
- Sensor Analytics (Export): Enable users to export sensor data in CSV, XLSX, or Google Sheets format for historical analysis.
- AI Crop Health Prediction Tool: Use generative AI as a tool to provide textual crop health predictions or alerts based on real-time data, and let the tool decide when to incorporate particular aspects of sensor reading data in its output. This is advisory only.
- Funding Opportunity Analyzer: Evaluate potential for investors to invest in particular areas.
- Live Monitoring Feeds: Share and evaluate media coming directly from wifi enabled field assets such as camera feeds for problem areas.

## Style Guidelines:

- Primary color: A fresh shade of green (#90EE90), symbolizing growth and nature.
- Background color: A desaturated, light green (#E0F8E0), creating a clean, calming backdrop.
- Accent color: A slightly more vibrant green (#7CFC00), drawing attention to important interactive elements.
- Modern card layout with interactive charts and dashboards for visualizing sensor data and AI predictions.
- Headline font: 'Space Grotesk', a sans-serif font, for a modern, tech-focused feel to the headers. Body font: 'Inter' for body text. Note: currently only Google Fonts are supported.
- Clear, intuitive icons related to agriculture, sensors, and data visualization to enhance usability.
- Subtle animations and transitions to enhance user engagement when updating data and displaying alerts.
